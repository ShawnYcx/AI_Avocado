Windows:

1. Launch command prompt on the project folder "AI_Avocado" or navigate 
to the project folder using the command prompt

2. to run the program:
   Type: "go <folder name>" e.g. "go phase_4"

//////////////////////////////////////////////////////////////////////
//////////////////////////////////////////////////////////////////////
//////////////////////////////////////////////////////////////////////

Mac OSX:

1. Launch terminal on the project folder "AI_Avocado" or navigate to 
the project folder using terminal
	e.g. "cd Desktop/AI_Avocado"

2. to run the program:
   type: "./go_<folder name>" e.g. "./go_phase4" or "./go_phase2"
